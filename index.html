API Key Usage
x

This tool uses the free LLM Groq, so users will need to obtain their own API key. The API key is stored in the browser and is not shared with other people or developer.

You can create an API key from this page and set it in the API key menu. If you want to update or delete your API key, set an empty string.

If you want to gain deep insights about the future, please choose Three Card. If you want a clear answer to a specific question, please choose Yes/No.
